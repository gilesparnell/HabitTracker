# HabitTracker

Single-user sobriety habit tracker PWA for vaping and drinking day counters.

- Versioning: bump the version and update `CHANGELOG.md` on every PR that ships code; the footer shows `vX.Y.Z (sha7)`.
- Migrations: SQL in `supabase/migrations/` is applied manually to the existing whole-life-challenge Supabase project, never automatically.
- Test command: `npx vitest run`.
- Plan: `docs/plans/2026-07-08-001-feat-sobriety-habit-tracker-pwa-plan.md`.
